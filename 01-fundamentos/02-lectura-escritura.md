# Lectura y Escritura de Información

## ¿Qué es la entrada y salida de datos?

Todo algoritmo o programa necesita comunicarse con el exterior para recibir información y mostrar resultados.

Esta comunicación se realiza mediante dos procesos fundamentales:

* **Entrada (Lectura):** recepción de datos.
* **Salida (Escritura):** presentación de resultados.

---

# Importancia

La entrada y salida de datos permiten:

* Interactuar con el usuario.
* Obtener información necesaria para resolver problemas.
* Mostrar resultados de forma comprensible.
* Dar utilidad práctica a los programas.

Sin entrada ni salida, un algoritmo no podría recibir información ni comunicar resultados.

---

# Entrada de datos (Lectura)

La lectura consiste en recibir información desde una fuente externa.

## Ejemplos

* Ingresar una edad.
* Escribir un nombre.
* Introducir una calificación.
* Leer datos desde un archivo.

---

# Tipos de entrada

| Tipo            | Descripción                           |
| --------------- | ------------------------------------- |
| Manual          | Datos ingresados por el usuario.      |
| Archivo         | Datos obtenidos desde archivos.       |
| Sistema externo | Datos provenientes de otros sistemas. |

---

# Salida de datos (Escritura)

La escritura consiste en mostrar o almacenar información generada por el algoritmo.

## Ejemplos

* Mostrar mensajes.
* Imprimir resultados.
* Generar reportes.
* Almacenar información en archivos.

---

# Tipos de salida

| Tipo                | Descripción                                 |
| ------------------- | ------------------------------------------- |
| Pantalla            | Información mostrada al usuario.            |
| Archivo             | Información almacenada para uso posterior.  |
| Dispositivo externo | Impresoras, pantallas u otros dispositivos. |

---

# Ejemplo conceptual

## Problema

Calcular el área de un rectángulo.

### Entrada

* Base
* Altura

### Proceso

* Área = Base × Altura

### Salida

* Área calculada

---

# Aplicaciones

La lectura y escritura de información están presentes en prácticamente todos los sistemas informáticos:

* Calculadoras.
* Sistemas bancarios.
* Aplicaciones móviles.
* Videojuegos.
* Sistemas de inventario.
* Software empresarial.

---

# Errores comunes

| Error                             | Descripción                                               |
| --------------------------------- | --------------------------------------------------------- |
| No solicitar los datos necesarios | El algoritmo no puede resolver el problema correctamente. |
| Solicitar información confusa     | El usuario puede ingresar datos incorrectos.              |
| No validar entradas               | Se aceptan datos inválidos.                               |
| No mostrar resultados             | El usuario no conoce el resultado obtenido.               |

---

# Conclusión

La lectura y escritura de información permiten que los algoritmos interactúen con su entorno. Gracias a ellas es posible recibir datos, procesarlos y comunicar resultados de manera efectiva.

---

# Resumen

| Concepto    | Idea principal                                          |
| ----------- | ------------------------------------------------------- |
| Entrada     | Recepción de datos.                                     |
| Lectura     | Obtención de información desde una fuente externa.      |
| Salida      | Presentación de resultados.                             |
| Escritura   | Comunicación de información al exterior.                |
| Importancia | Permite la interacción entre el algoritmo y su entorno. |
