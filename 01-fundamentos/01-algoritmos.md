# Algoritmos

## ¿Qué es un algoritmo?

Un **algoritmo** es un conjunto de pasos ordenados, finitos y precisos que permiten resolver un problema o realizar una tarea específica.

En programación, los algoritmos representan la lógica que debe seguirse para obtener un resultado determinado.

---

# Importancia de los algoritmos

Los algoritmos constituyen la base de la programación porque permiten:

* Analizar problemas de manera estructurada.
* Diseñar soluciones antes de escribir código.
* Organizar las acciones necesarias para alcanzar un objetivo.
* Facilitar la comprensión y mantenimiento de los programas.

Antes de programar, es necesario definir claramente el algoritmo que resolverá el problema.

---

# Problema y solución

Todo algoritmo surge para resolver un problema.

### Problema

Situación que requiere una solución para alcanzar un objetivo.

### Solución

Conjunto de acciones que permiten resolver el problema.

Por ejemplo:

**Problema:** Calcular el promedio de tres calificaciones.

**Solución:**

1. Leer las tres calificaciones.
2. Sumar las calificaciones.
3. Dividir la suma entre tres.
4. Mostrar el resultado.

---

# Modelo Entrada → Proceso → Salida

La mayoría de los algoritmos siguen una estructura básica:

```text
Entrada → Proceso → Salida
```

## Entrada

Datos necesarios para resolver el problema.

### Ejemplos

* Edad de una persona.
* Cantidad de productos.
* Calificaciones de un estudiante.

## Proceso

Operaciones realizadas sobre los datos de entrada.

### Ejemplos

* Sumas.
* Restas.
* Comparaciones.
* Cálculos matemáticos.

## Salida

Resultados obtenidos después de procesar la información.

### Ejemplos

* Promedio calculado.
* Total a pagar.
* Mensaje de aprobación o reprobación.

---

# Ejemplo de algoritmo

## Problema

Calcular la suma de dos números.

### Entrada

* Número A
* Número B

### Proceso

* Sumar A + B

### Salida

* Resultado de la suma

### Algoritmo

1. Leer A.
2. Leer B.
3. Calcular suma = A + B.
4. Mostrar suma.

---

# Características de un algoritmo

Un algoritmo debe cumplir ciertas características.

| Característica | Descripción                                                          |
| -------------- | -------------------------------------------------------------------- |
| Finito         | Debe terminar después de ejecutar una cantidad determinada de pasos. |
| Preciso        | Cada instrucción debe estar claramente definida.                     |
| Ordenado       | Los pasos deben seguir una secuencia lógica.                         |
| Efectivo       | Debe resolver correctamente el problema.                             |
| Determinístico | Con las mismas entradas produce siempre las mismas salidas.          |

---

# Algoritmo y programa

Aunque suelen relacionarse, no son lo mismo.

| Algoritmo                                            | Programa                                               |
| ---------------------------------------------------- | ------------------------------------------------------ |
| Describe la solución lógica.                         | Implementa la solución en un lenguaje de programación. |
| Es independiente del lenguaje.                       | Depende del lenguaje utilizado.                        |
| Puede escribirse en lenguaje natural o pseudocódigo. | Se escribe utilizando código fuente.                   |

Un mismo algoritmo puede implementarse en distintos lenguajes como C++, Python o Java.

---

# Formas de representar algoritmos

Los algoritmos pueden representarse de diferentes maneras.

## Lenguaje natural

Describe los pasos utilizando texto común.

### Ejemplo

1. Leer dos números.
2. Sumarlos.
3. Mostrar el resultado.

## Pseudocódigo

Representa los pasos mediante una estructura similar a un programa.

## Diagramas de flujo

Representan gráficamente la secuencia de acciones mediante símbolos.

## Código fuente

Implementación del algoritmo en un lenguaje de programación.

# Aplicaciones de los algoritmos

Los algoritmos están presentes en prácticamente todos los sistemas informáticos:

* Aplicaciones móviles.
* Sistemas bancarios.
* Videojuegos.
* Redes sociales.
* Comercio electrónico.
* Inteligencia artificial.
* Sistemas de gestión empresarial.

---

# Conclusión

Los algoritmos constituyen la base de la programación y de la resolución de problemas mediante computadoras. Antes de desarrollar un programa es fundamental diseñar una solución clara, organizada y eficiente.
