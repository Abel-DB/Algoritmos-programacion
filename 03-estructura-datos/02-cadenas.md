# Cadenas

## ¿Qué es una cadena?

Una **cadena** es una secuencia de caracteres almacenados y tratados como una única unidad de información.

Las cadenas permiten representar texto dentro de un algoritmo o sistema informático.

### Ejemplos

```text
Hola
Programación
Juan Pérez
2025
```

---

# Importancia

Las cadenas permiten:

* Almacenar texto.
* Mostrar mensajes.
* Representar nombres y descripciones.
* Procesar información textual.
* Validar datos ingresados.

---

# Conceptos fundamentales

| Concepto | Descripción                                   |
| -------- | --------------------------------------------- |
| Cadena   | Conjunto de caracteres.                       |
| Carácter | Unidad individual de información textual.     |
| Longitud | Cantidad de caracteres que posee una cadena.  |
| Índice   | Posición de un carácter dentro de una cadena. |

---

# Relación con ASCII

Cada carácter de una cadena posee una representación numérica asociada mediante el código ASCII.

### Ejemplo

```text
Cadena: ABC

A → 65
B → 66
C → 67
```

Gracias a esta representación, los sistemas informáticos pueden almacenar y procesar texto.

---

# Relación con las tablas

Conceptualmente, una cadena puede considerarse una colección ordenada de caracteres.

### Ejemplo

```text
Cadena: Hola

Índice:     0   1   2   3
Carácter:   H   o   l   a
```

Cada carácter ocupa una posición específica dentro de la cadena.

---

# Operaciones comunes sobre cadenas

## Concatenación

Consiste en unir dos o más cadenas para formar una nueva.

### Ejemplo

```text
Nombre: Juan
Apellido: Pérez

Resultado:

Juan Pérez
```

---

## Longitud

Representa la cantidad de caracteres contenidos en una cadena.

### Ejemplo

```text
Cadena: Hola

Longitud: 4
```

---

## Comparación

Permite determinar si dos cadenas son iguales o diferentes.

### Ejemplo

```text
Hola = Hola     → Verdadero
Hola = hola     → Falso
```

---

## Búsqueda

Permite localizar caracteres o palabras dentro de una cadena.

### Ejemplo

```text
Cadena: Programación

Buscar: "gram"

Resultado: Encontrado
```

---

# Representación conceptual


---

# Aplicaciones

Las cadenas se utilizan en:

* Nombres de usuarios.
* Direcciones.
* Mensajes.
* Contraseñas.
* Sistemas de búsqueda.
* Procesamiento de texto.
* Formularios y registros.

---

# Errores comunes

| Error                                                  | Descripción                                                                  |
| ------------------------------------------------------ | ---------------------------------------------------------------------------- |
| Confundir carácter con cadena                          | Un carácter es una unidad individual; una cadena contiene varios caracteres. |
| Ignorar mayúsculas y minúsculas                        | Pueden representar valores distintos.                                        |
| Acceder a posiciones inexistentes                      | Produce errores en el procesamiento.                                         |
| Suponer que todas las cadenas tienen la misma longitud | Cada cadena puede contener diferente cantidad de caracteres.                 |

---

# Conclusión

Las cadenas permiten representar y manipular información textual dentro de algoritmos y sistemas informáticos. Constituyen una de las estructuras de datos más utilizadas debido a la gran cantidad de aplicaciones relacionadas con el manejo de texto.
