# Algoritmos y Programación

Repositorio de estudio y práctica de algoritmos, lógica de programación y C++.

El objetivo de este repositorio es documentar de forma organizada el proceso de aprendizaje, incluyendo teoría, ejercicios resueltos, pseudocódigo, diagramas de flujo, pruebas de escritorio e implementaciones en C++.

# Metodología Utilizada

Cada ejercicio sigue la siguiente estructura:

1. Enunciado
2. Análisis del problema
3. Diseño de la solución

   * Entradas
   * Proceso
   * Salidas
   * Secuencia lógica
4. Pseudocódigo
5. Diagrama de flujo
6. Prueba de escritorio
7. Implementación en C++
