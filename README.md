# Apuntes de Algoritmos y Programación

Este repositorio contiene apuntes organizados sobre los fundamentos de la programación. El contenido está estructurado de forma progresiva para facilitar el estudio, la comprensión y el repaso de los distintos temas.

---

## Contenido

### Tema 01: Fundamentos

* Algoritmos
* Diagramas de flujo
* Resolución de problemas

### Tema 02: Proceso de Programación

* Edición
* Compilación
* Enlazado

### Tema 03: Datos

* Lectura y escritura de información
* Variables
* Tipos de datos
* Operadores

### Tema 04: Representación de Datos

* Código ASCII

### Tema 05: Control

* Instrucciones de control

### Tema 06: Estructuras Básicas

* Arreglos (tablas)
* Cadenas (strings)

### Tema 07: Estructuras Compuestas

* Estructuras (registros)

### Tema 08: Herramientas

* Macros

---

## Objetivo

* Organizar los contenidos de la materia
* Facilitan el estudio y la revisión
* Comprender la lógica de programación
* Servir como material de apoyo

---

## Metodología

Los apuntes incluyen:

* Explicación teórica
* Diagramas de flujo
* Pruebas de escritorio
* Ejemplos conceptuales

---

## Estado

El repositorio se encuentra en desarrollo y se actualiza conforme se avanza en los temas.
