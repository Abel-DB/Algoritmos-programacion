# 2. Análisis

## Entradas

### Ejemplo

```text

```

---

## Proceso

Operaciones o transformaciones necesarias para resolver el problema.

### Ejemplo

```text
Calcular el área.
Verificar si un número es primo.
Obtener el promedio.
```

---

## Salidas

Resultados que debe producir el algoritmo.

### Ejemplo

```text
Área calculada.
Promedio final.
Mensaje de aprobación.
```

---

## Restricciones

Condiciones que deben cumplirse.

### Ejemplo

```text
edad >= 0
nota >= 0 y nota <= 100
```
