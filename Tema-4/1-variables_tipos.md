## ¿Qué es una variable?

Una **variable** es un espacio en memoria que se utiliza para almacenar datos.

Su valor puede cambiar durante la ejecución del programa.

---

## Ejemplo sencillo

Si queremos guardar un número:

* Variable: A
* Valor: 5

A = 5

---

## Características de una variable

* Tiene un **nombre**
* Tiene un **tipo de dato**
* Tiene un **valor**

---

## Tipos de datos

Los tipos de datos indican qué tipo de información se puede almacenar.

---

### Tipos básicos

* **Entero (int):** números sin decimales
  Ej: 5, -3, 10

* **Real (float):** números con decimales
  Ej: 3.5, -2.7

* **Carácter (char):** un solo carácter
  Ej: 'A', 'b', '1'

* **Cadena (string):** conjunto de caracteres
  Ej: "Hola", "Juan"

* **Booleano (bool):** verdadero o falso
  Ej: true, false

---

## Uso de variables en un algoritmo

**Problema:** Sumar dos números

---

### Algoritmo

1. Leer A
2. Leer B
3. Suma = A + B
4. Mostrar Suma

---

## Importancia de los tipos de datos

* Evitan errores
* Permiten usar correctamente la memoria
* Definen qué operaciones se pueden realizar

---

## Ejemplos de uso incorrecto

* Guardar texto en una variable numérica
* Usar decimales en variables enteras

---

## Resumen

* Una variable almacena datos
* Tiene nombre, tipo y valor
* Los tipos de datos definen qué se puede guardar
* La prueba de escritorio permite verificar el algoritmo
