# Cadenas (Strings)

---

## ¿Qué es una cadena?

Una **cadena** es una secuencia de caracteres que pueden ser:

* Letras
* Números
* Símbolos

Ejemplos:

```
"Hola"
"Juan123"
"@correo.com"
```

---

## ¿Cómo funciona una cadena?

Una cadena funciona como un **arreglo de caracteres**, donde cada posición tiene un índice.

Ejemplo:

```
Posición:   0   1   2   3
           ┌───┬───┬───┬───┐
Cadena:    │ H │ O │ L │ A │
           └───┴───┴───┴───┘
```

* cadena[0] → H
* cadena[3] → A

---

## Características

* Es una secuencia ordenada de caracteres
* Se accede mediante índices
* El índice comienza en 0
* Tiene una longitud (tamaño)
* Permite representar texto

---

## Operaciones básicas (concepto)

Con las cadenas se pueden realizar diversas operaciones:

* Recorrer los caracteres
* Obtener la longitud
* Comparar cadenas
* Unir (concatenar)
* Acceder a posiciones específicas
* Modificar contenido

---

## Relación con ASCII

Cada carácter de una cadena tiene una representación numérica interna según el código ASCII.

Ejemplos:

* 'A' → 65
* 'a' → 97

Esto permite comparar y manipular texto a nivel interno.

---

## Errores comunes

* Pensar que los índices comienzan en 1
* Confundir mayúsculas con minúsculas
* Acceder a posiciones fuera del tamaño
* Comparar cadenas incorrectamente

---

## Aplicaciones

* Nombres de usuarios
* Correos electrónicos
* Validación de datos
* Procesamiento de texto

---

## Importancia

Las cadenas son fundamentales para trabajar con información textual en cualquier programa.

---

## Conclusión

Las cadenas permiten representar y manipular texto de forma estructurada, siendo una de las estructuras más utilizadas en programación.

---

## Resumen

* Una cadena es un conjunto de caracteres
* Funciona como un arreglo
* Se accede por índices (desde 0)
* Permite manejar texto de diferentes formas
